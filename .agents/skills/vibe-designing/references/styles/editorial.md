# Editorial style

- Use a refined, premium editorial direction: generous whitespace, strong hierarchy, exacting typography, and a calm sense of expense.
- Use `0px` corner radii by default everywhere. Only introduce radius when it has a clear product, brand, or usability reason.
- Default to light-mode editorial layouts unless the user requests dark mode or the subject matter strongly calls for it. This should guide contrast and atmosphere, not restrict the palette to pale colors or low-saturation shades.
- Add subtle motion, animation, and transitions that reinforce hierarchy, state changes, and flow.
- Prioritize best-in-class micro-interactions that feel precise, restrained, and tactile.
- Avoid print-production jargon, such as `folio`, `vol`, `§ II`, `MMXXVI`, `masthead`, `deck`, `slug`, `proof`, or `spread`, unless the product explicitly concerns print workflows.
